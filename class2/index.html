Object Creation

Let's create our very own object

Objects

Objects let us store a collection of properties.

As you can see they are really easy to create.


var about_me = {
hometown: 'Nitro, WV',
hair: 'Brown',
likes: ['knitting', 'code', 'music', 'woodworking','screen printing'],
birthday: {month: 3, day: 3},
};

Accessing the Data In Objects

You can retrieve values using "dot notation"


var about_me = {
hometown: 'Nitro, WV',
hair: 'Brown',
likes: ['knitting', 'code', 'music', 'woodworking','screen printing'],
birthday: {month: 3, day: 3},
};
var my_hometown = about_me.hometown;

Or using "bracket notation" (like arrays)


var my_hair = about_me['hair'];

Changing Objects

You can use dot or bracket notation to change property values


var about_me = {
hometown: 'Nitro, WV',
hair: 'Brown',
likes: ['knitting', 'code', 'music', 'woodworking','screen printing'],
birthday: {month: 3, day: 3},
};
about_me.hair = 'blue';

Add new properties


          about_me.vehicle = 'motorcylce';

Or delete them


          delete about_me.vehicle;

Objects are Very Portable

};

function describePerson(person) {
console.log('This person is ' + person.age + ' years old with ' + person.hair + ' hair.');
}

describePerson(about_me);

Object Methods

When objects do things

Object methods

Objects can also hold functions.


var sager = {
age: 39,
eyes: 'hazil',
talk: function() {
  console.log('nerd stuff');
},
eat: function(food) {
  console.log('Yum, I love ' + food);
}
};

Call object methods using dot notation:


sager.talk();
sager.eat('Kale');

Object Methods are very convenient

And you use them ALL the time.

  • console.log() - Object Method
  • document.getElementById() - Object Method
  • window.confirm() - Object Method
