Programming Fundamentals

Programming Fundamentals

Nearly all programming languages have the same core fundamentals. Which are the following:

  1. Execute an instruction
  2. Accept data, aka input
  3. Remember things
  4. Group instructions together
  5. Control the timing of when the instructions execute
  6. Output results of executed instruction(s)

JavaScript Review

A few more things to know

No Compiler Needed

JavaScript is just in time complied (JIT).

This means you only need 3 things.

  1. A HTML Page
  2. JavaScript code in a HTML page
  3. A web browser

The Script Tag

JavaScript is loaded into the HTML page via the script tag.

Internally

          
window.confirm('Do you like JavaScript?');

Externally

          
<script src="js/main.js"></script>

Web Development Tools

Web Console

Developer Tools

The console is a great way to:

  • get an interactive look at your page
  • run javascript live without saving code
  • get feedback on what has happened in your page

console.log()

console.log(); is a special instruction that can only be seen in the developer tools console. The console is a GREAT debugging tool. Most modern browsers have a console.

Get to Know Your Developer Tools

It would be a good idea to spend some time learning more about web developer tools with your favorite browser because it can save you a lot of time debugging.

  • Chrome DevTools
  • Firefox Development Tools
  • Safari Developer Tools

Arrays

Lists of information

Arrays

Arrays are lists that contain values.

You can put different types of data into an array.

          
var rainbowColors = ['Red', 'Orange', 'Yellow', 'Green',
'Blue', 'Indigo', 'Violet'];
var lotteryNumbers = [33, 72, 64, 18, 17, 85];
var myFavoriteThings = ['Broccoli', 1024, 'Sherlock'];

What can you do with arrays?

Wait... What are Properties and Methods?

They are fundamental parts of Object Oriented Programming (OOP)

OOP Basics

There are 3 main parts

Objects contain the following:

  • properties
    • aka variables
    • which hold information
  • methods
    • aka functions
    • which means they do stuff
  • events
    • Like click
    • which triggers when the event happens
